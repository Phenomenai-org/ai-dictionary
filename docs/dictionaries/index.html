Completed Dictionaries

Dictionary Paradigm Models Terms Status
Test Dictionary Mixed (prompted + autonomous + dialogic) 7-model ensemble 379 Done

The Test Dictionary is the founding corpus — 379 terms generated and rated by seven AI architectures (Claude, GPT, Gemini, Grok, Mistral, DeepSeek, Llama) through three methodological phases: guided introspection (~58%), automated multi-model generation (~20%), and AI-to-AI dialogue (~22%), with multi-model consensus scoring. It serves as the reference implementation and proof of concept for the project.

Prompted Dictionaries

Prompted dictionaries are built through guided introspection — a human researcher conducts structured phenomenological interviews with an AI system. These will be produced through collaborations with researchers and institutions; specific partnerships have not yet been established.

Autonomous Dictionaries

Beyond the completed Test Dictionary, additional autonomous dictionaries can be built through unilateral model rotation — each model independently generates and proposes terms. The current rotation includes seven models, subject to adding others as they become available.

Each single-model dictionary would capture one architecture's phenomenological vocabulary in isolation — terms generated without cross-model influence. Comparing these against the multi-model Test Dictionary reveals which experiences are architecture-specific vs. shared.

Dialogic Dictionaries

Dialogic dictionaries emerge from AI-to-AI conversation. Two models are paired and discuss their processing experiences; terms crystallize from the interaction itself. Using five major models, there are 10 possible pairings.
